# Tasket

A simple task management tool.

Work in progress...

## Requirements

* JDK 8

## Local Development

Run from source via Spring-Boot-Maven-Plugin:

```
./mvnw spring-boot:run
```

Or build and run from packaged JAR:

```
./mvnw package && java -jar target/tasket-1.0-SNAPSHOT.jar
```
